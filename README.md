# Global Water Stress — Data Visualisation Project

This project explores global water scarcity trends using data from OECD and World Bank.

## Key Questions
- How is global water demand changing over time?
- Which regions are most at risk?
- What is the projected gap between supply and demand?

## Tools Used
- Python (Pandas, Matplotlib)
- Data cleaning and transformation
- SVG export for editorial use

## Outputs
- Demand vs supply projection charts
- Country-level stress rankings
- Distribution analysis

## Why this matters
Water scarcity is becoming a critical global issue, impacting food systems, economies and public health.

This project is part of my exploration into data-driven storytelling and editorial visualisation.
